How it works

  • Shops request a higher plan from Upgrade Plan.
  • Super admin approves under Upgrade Requests.
  • On approval, plan is applied and a subscription invoice is generated with CGST + SGST lines.
  • Intrastate Indian GST: CGST + SGST = total GST (e.g. 9% + 9% = 18%).
